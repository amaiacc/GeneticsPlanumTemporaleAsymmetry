Scripts to combine results from multiple different sources.

-----------------------------------------
Heritability
-----------------------------------------
# ./GCTA/
Scripts to run REML analyses for PT phenotypes.
# ./ldsc/
Scripts to run heritability and genetic correlation analyses using ldsc.
# ./sumher/
Scripts to run heritability and genetic correlation analyses using sumher.

-----------------------------------------
Association
-----------------------------------------
# ./bgenie/
Scripts for GWAS, filtering and visualization.
# ./easyStrata/
Scripts to create Miami plots... need to remove chrX so that they work.
# ./phesant/
Scripts to run phenome-wide association analyses for lead SNPs.
# ./lead_snps/
Scripts to perform quality checks and downstream analyses on lead SNPs.

Gene and gene-set
-----------------------------------------
# ./magma/
Scripts to run gene and gene-set enrichment analyses.
